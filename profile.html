My Coding Jouney

Currently, I am between jobs but I am using this time as an opportunity to learn to code! My objective (or my "why") in joining this bootcamp is to be able to change careers, secure a remote position, and enjoy my days of coding and being outdoors. My ideal dream job would be creating/managing websites or developing parts of websites for larger companies. I have really been enjoying the technical and problem solving of my coding journey so far.

Prior to this course I have worked through some Freecodecamp/Codecademy HTMT, CSS, and JavaScript courses and used chatGPT to help me figure out how to push code to GitHub and use their pages feature to host my first coding experiments. Additionally, I took an intro to QA automation course which introduced me to some really cool software (cypress) and was a great crashcourse into javascript and the CLI.

Resources:

  • Freecodecamp
  • Codecademy
  • QA automation course

I'm looking forward to this Web Development Fundamentals course and the following Full Stack Web Mobile Development course I'm taking immediately after this one. What I am most looking forward to is seeing it all come together. For example: Making a form in html, styling it in css, and then using an assortment of languages and systems for the back end? I don't know the specifics yet but I am excited to find out!
