Installation Instruction

  • In Visual Studio Code, click the Extensions icon in th left column toolbar.
  • In the search bar that appears, type HTML Preview.
  • Select the HTML Preview extension that is published by George Oliviera.
  • In the page for Live Server that appears, click the green Install button.
    •

    Usage Instructions

    The extension can be activated in two ways

    • Toggle Preview - ctrl+shift+v or cmd+shift+v

    • Open Preview to the Side - ctrl+k v or cmd+k

    NOTE: some embedded elements (like YouTube videos) wont work with this extension
